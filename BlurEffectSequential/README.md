# Blur effect

## Build and run

```
make && ./bin/blur-effect img/favicon.png out/testing.png 3

# Make options
make clean # Delete objects folder
make fullclean # Delete objects and bin folder
```

## Folder structure

- `docs`: Documentation
- `img`: Images to process
- `src`: Source code
- `bin`: Executable folder
- `obj`: C objects folder
- `lib`: Libraries
- `out`: Output folder
